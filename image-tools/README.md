# Image Tools

Fullstack application for image optimization and formatting.

## Features

- Client-Server architecture
- CORS requests
- Form handling

## How to run

### Client

```shell
cd client
npm install
npm run build
npm preview
```

You can also start client in development mode:

```shell
npm run dev
```

### Server

```shell
cd server
npm install
npm run build
npm start
```
